# ALX Travel App 0x00

## Project Overview
This project is a continuation of the **ALX Travel App**, designed to facilitate and build upon prior Django models, serializers, and management commands.

## Project Objectives
1. Duplicate the base project ('alx_travel_app') into a new repository ('alx_travel_app_0x00').
2. Define database models inside 'listings/models.py'.
3. Create serializers in 'listings/serializers.py'.
4. Implement management commands for data seedling in 'listings/management/commands/seeds.py'.
5. Run and test the commands.
